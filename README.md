# Demand_Forecasting_Data
This projects analyze demand of the custumer data.The goal is to provide insights that can help improve business decisions.

--

## Business Problem
The company wants to understand:
* what is the demand of the season?
* Calculate the StockPressure?
* Calculate the Competation Pircing?

  --

  ## Dataset
- Source: Kaggle
- Records: 100+
- Features:
  Date
  Store_ID
  Product_ID
  Category
  Region
  Inventory_Level
  Units_Sold
  Units_Ordered
  Price
  Discount
  Weather_Condition
  Promotion
  Competitor_Pricing
  Seasonality
  Epidemic
  Demand




--

- SQL
- Microsoft Excel
- Power BI
- Python

## Project Workflow
1. Imported the dataset.

5. Created KPIs and visualizations.
6. Built an interactive Power BI dashboard.
